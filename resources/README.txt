================================================================================
  MeeMee Screener ポータブル版
================================================================================

【使い方】

1. このフォルダを任意の場所に配置してください
   （例: C:\Tools\MeeMeeScreener）

2. MeeMeeScreener.exe をダブルクリックして起動してください

3. 初回起動時は自動的にセットアップが行われます


【初回起動時の注意】

- WebView2ランタイムが必要です
  未インストールの場合は自動的にダウンロード・インストールされます

- データフォルダが自動的に作成されます


【データの保存場所】

通常モード（デフォルト）:
  %LOCALAPPDATA%\MeeMeeScreener\data
  （例: C:\Users\YourName\AppData\Local\MeeMeeScreener\data）

ポータブルモード:
  実行ファイルと同じフォルダ内の data サブフォルダ
  
  ポータブルモードを有効にするには:
  1. MeeMeeScreener.exe と同じフォルダに portable.txt を作成
  2. アプリケーションを起動
  
  ※ USBメモリなどに入れて持ち運ぶ場合に便利です


【システム要件】

- Windows 10 以降（64bit）
- .NET Framework 4.8 以降
- WebView2 ランタイム（自動インストール）


【アンインストール】

1. このフォルダを削除
2. データも削除する場合は以下のフォルダも削除:
   - %LOCALAPPDATA%\MeeMeeScreener （通常モードの場合）
   - または、ポータブルモードで作成された data フォルダ


【更新方法】

1. 新しいバージョンのZIPファイルをダウンロード
2. 古いフォルダを削除（dataフォルダは残す）
3. 新しいZIPを展開
4. データフォルダは自動的に引き継がれます


【トラブルシューティング】

■ アプリケーションが起動しない
  - WebView2ランタイムが正しくインストールされているか確認
  - ウイルス対策ソフトがブロックしていないか確認
  - 管理者権限で実行してみる

■ データが表示されない
  - データフォルダの場所を確認
  - ポータブルモードと通常モードが混在していないか確認

■ エラーメッセージが表示される
  - ログファイルを確認: %LOCALAPPDATA%\MeeMeeScreener\logs


【サポート】

問題が解決しない場合は、ログファイルと共にお問い合わせください。
