# Event Badges

## 目的

決算日、権利落日、その他主要イベントを小さなバッジとして一貫表示する。

## 表示対象

- 決算日
- 権利落日
- 主要な適時開示イベント
- 補助的な貸借イベント

## 真実ソースの優先順位

- 法定財務: EDINET
- 適時開示: TDNET
- 権利付き / 権利落ち系: JPX
- J-Quants 等: 補助取得層

## 予定 / 実績 / 変更

- 予定日は予定として残す
- 実績が出たら実績を優先して表示する
- 変更が入った場合は、変更済みであることを隠さない
- 予定と実績がズレたら、予定を上書きして消さない

## 画面上の扱い

- バッジは一覧と詳細の両方で使える
- 小さい領域で読みやすい短いラベルにする
- 重大イベントだけを先に出し、細かい分類は増やしすぎない
