## 次にやること

### 📊 保有銘柄を表示するには

1. **取引履歴CSVを準備**
   - 楽天証券またはSBI証券から取引履歴CSVをダウンロード

2. **フロントエンドでインポート**
   - ブラウザで http://localhost:3000 を開く
   - 「保有/履歴」タブをクリック
   - CSVファイルをアップロード

3. **結果確認**
   - 保有銘柄が一覧表示されます
   - 売-買の数量が表示されます
   - 取引開始日が表示されます

---

### 📅 決算日・権利落ち日を表示するには

1. **イベント更新を実行**
   - 任意の銘柄の詳細画面を開く
   - 画面上部の「イベント更新」ボタンをクリック
   - 30秒〜1分待つ

2. **結果確認**
   - 銘柄詳細画面に決算予定日が表示されます
   - 権利落ち日が表示されます
   - 「イベント最終更新」の日時が更新されます

## トラブルシューティング

### 「保有銘柄はありません」と表示される

**原因:** 取引履歴がインポートされていません

**解決方法:**
```bash
# 診断スクリプトで状態を確認
python diagnose_issues.py

# 取引履歴CSVをフロントエンドからアップロード
```

### 「更新失敗」と表示される

**原因1:** インターネット接続の問題
- ネットワーク接続を確認してください
- JPXのウェブサイト (jpx.co.jp) にアクセスできるか確認

**原因2:** ライブラリ不足 (修正済み)
- すでに修正されています
- 念のため確認: `python diagnose_issues.py`
